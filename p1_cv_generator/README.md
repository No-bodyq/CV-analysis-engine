## Setup Instructions

1. Clone the repository
2. Install Python 3.10+

3. Create virtual environment:
   python -m venv venv

4. Activate it:
   .\venv\Scripts\Activate.ps1

5. Install dependencies:
   pip install -r requirements.txt

6. Create .env file:
   Copy from .env.example and add your API key

7. Run the project:
   python app.py
